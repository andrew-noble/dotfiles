# nvim cheatsheet

## operators + motions (skip visual mode)

`d5j` тАФ delete 5 lines down  
`y3w` тАФ yank 3 words  
`c$` тАФ change to end of line  
`d$` тАФ delete to end of line  
`dap` тАФ delete a paragraph  
`yip` тАФ yank inner paragraph  

## commenting

`gcc` тАФ toggle current line  
`gc5j` тАФ toggle 5 lines down  
`gc` тАФ toggle selection (visual mode)  

## indenting

`>>` / `<<` тАФ indent / dedent current line  
`>5j` тАФ indent 5 lines down  
`=ap` тАФ auto-indent a paragraph
